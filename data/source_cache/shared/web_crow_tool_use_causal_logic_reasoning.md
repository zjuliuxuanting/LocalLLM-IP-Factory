# 乌鸦工具使用的因果逻辑推理

> Source: https://html.rhhz.net/DSJYJ/20170408.htm

## 核心研究

马东东, 裴树文 (2017), "旧石器时代早期石器技术与人类认知能力关系研究的回顾与探讨" — 旧石器时代早期石器技术与人类认知能力的关系。王琳 et al. (2020), "鸦科鸟类认知研究回顾" — 鸦科鸟类认知研究的系统性综述。

## 实验发现

乌鸦在工具使用过程中展现出因果逻辑推理能力，能够理解工具与目标之间的因果关系，并据此调整自己的行为策略。

## 认知能力

乌鸦的工具使用不仅仅是本能反应，而是基于对物理世界的理解和预测。这种能力在进化上具有重要意义。
