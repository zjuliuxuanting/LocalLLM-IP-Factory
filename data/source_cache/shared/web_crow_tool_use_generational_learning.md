# 乌鸦工具使用的代际学习

> Source: https://www.researchgate.net/profile/Zhongqiu-Li-3/publication/346887642_A_Review_of_Cognitive_Studies_on_Corvids/links/5fd1bfb892851c00f8623ccc/A-Review-of-Cognitive-Studies-on-Corvids.pdf

## 核心研究

王琳, 罗云超, 李忠秋 (2020), "鸦科鸟类认知研究回顾" — 鸦科鸟类认知研究的系统性综述。

## 实验发现

乌鸦能够通过社会学习掌握工具制造技能，并将这些技能传递给后代。这种代际学习能力在鸟类中极为罕见，展示了乌鸦复杂的社会结构和文化传承机制。

## 社会学习机制

年轻乌鸦通过观察成年个体的行为来学习工具使用技巧，这种学习方式类似于人类的文化传递过程。
