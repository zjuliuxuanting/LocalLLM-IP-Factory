# 宠物按钮'传家宝'：新猫到家第一天，旧猫用按钮把前任糗事全抖了

> Source: https://furwise.app/ja/blog/multi-cat-household/
**多頭飼いガイド｜新しい猫の迎え方・縄張り管理・対立の見分け方** (Furwise, 2026)

猫是 facultatively social（选择性社交）动物，在适当条件下可以形成社会群体，但并非必须如此。室内猫的困境在于它们没有选择权——被迫与未达成共识的同伴共享空间。
新猫引入的关键因素：
- 先住猫的经验：曾与其他猫和谐共处的猫比长期独居的猫更容易接受新成员
- 年龄和活动量平衡：幼猫与老年猫组合往往更困难，同窝幼猫是最顺利的配对之一
- 性格匹配：自信从容的猫比胆小焦虑的猫更容易接受新入者

**阶段式介绍法（AAFP/ISFM指南）：**
1. 完全隔离期（至少3天）：新猫安置在独立房间
2. 气味交换（第4-7天）：用毛巾擦拭新猫脸颊，让先住猫闻；交换睡垫和玩具
3. 网格门见面（第8-14天）：每次不超过5分钟，每天2-3次
4. 空间轮流使用（第15-21天）：交换空间1-2小时后归位

**多猫家庭相处指南：深度解析领地竞争、霸凌行为与费洛蒙减压实战案例** (Petflic, 2026)

多猫家庭的核心资源分配标准（N+1原则）：
| 资源种类 | 配置数量 | 布置战术 |
|---|---|---|
| 砂盆 | 猫口数+1 | 严禁并排，需设置于不同房间 |
| 饮水处 | 2处以上 | 远离食盆，设置于动线转角 |
| 进食点 | 每只猫独立 | 视线不交会（背对背或高低差） |
| 垂直躲藏点 | 猫口数×2 | 高处猫跳台，确保有双向出口 |

**隐形霸凌行为识别：**
- 资源封锁（Blocking）：强势猫安静地坐在通往砂盆、水碗或猫窝的必经之路上
- 眼神定格（Staring）：长时间紧盯着另一只猫，是极具威胁性的挑战信号
- 肚皮不着地：弱势猫长期处于戒备状态，不敢深层睡眠

**多猫家庭总打架？一眼看穿等级关系，5招化解地盘冲突！** (猫百科全书, 2025)

猫咪社会结构的独特之处：非线性支配关系——支配权可能因资源、空间或时间而变化。A猫在食盆旁占优势，B猫在猫爬架上占优势，没有绝对的"Alpha"概念。
**五大冲突根源：**
1. 资源争夺（食物、水、猫砂盆）
2. 休息区争霸战（窗台观景位、高处休息区）
3. 人类注意力分配不均
4. 领土边界纠纷
5. 新成员引入引发的领地危机

**关键发现：**
- 猫咪是选择性社交动物，被迫共享空间会引发慢性压力
- N+1资源分配原则是预防冲突的核心策略
- 阶段式介绍法（至少21天）可显著降低新猫引入的冲突风险
- 垂直空间和资源分散化是化解地面领地纷争的关键
