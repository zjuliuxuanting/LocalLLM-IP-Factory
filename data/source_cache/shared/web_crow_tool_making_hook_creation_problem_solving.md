# 乌鸦工具使用：从简单到复杂的技术迭代

> Source: https://wires.onlinelibrary.wiley.com/doi/abs/10.1002/wcs.1322

## 核心研究

AH Taylor & RD Gray (2014), "Is there a link between the crafting of tools and the evolution of cognition?" — 工具制作与认知进化之间是否存在联系？

## 实验发现

乌鸦能够制造钩状工具，通过弯曲树枝或草茎来钩取昆虫。这种能力展示了乌鸦对物理特性的理解和应用能力。

## 问题解决

乌鸦在工具使用过程中展现出灵活的问题解决能力，能够根据不同环境调整工具设计和使用方法。
